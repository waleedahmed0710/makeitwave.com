Wave® Multi-Room™ Link - V1 Hardware

See below the hardware specifications for the Wave® Home Hub. Learn more by reading this products User Guide

Specifications

  • 10/100 Ethernet Port for quick connection to internet
  • Micro SD port for loading operating system and storing data
  • Broadcom BCM2837 Arm7 Quad Core Processor running at 900MHz & 1GB RAM
  • 4 x USB 2 ports (1.2Amp Output)
  • Full size HDMI - Stream and watch Hi-definition video output at 1080P
  • 40pin extended GPIO - Touch screen display, other accessories etc
  • Combined 4-pole jack for connecting your stereo audio out and composite video out
  • 2Amp Micro USB Power Port - Includes Advanced Power Management
