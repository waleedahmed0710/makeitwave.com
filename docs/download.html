Insert Memory

Download the latest version of WaveOS (see below) and copy to your Mico SD Card using a tool like Etcher. To use this tool just select the WaveOS Image (dowloadable above) and hit "Flash!". It takes less than 1 minute for WaveOS to install to the SD Card. We suggest a Sandisk (32Gb or higher).

Connect Device

Attach the ethernet Cable to Wave and the other end into your internet router. Then connect the power cord. The system will boot up & use the internet to autonomously set itself up. This process takes 5 - 10 minutes.

Access Menu

Using your smartphone or computer, connect to the Wi-Fi (SSID:Wave-Hotspot/ Password: makeitwave). Once connected enter into your web-browser: "192.168.42.1" or "http://wave/". Then follow the onscreen instructions
