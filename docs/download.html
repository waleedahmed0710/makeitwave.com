Software Editions

Software Editions determin what function/ role your device will perform day to day. When you first use WaveOS, it will present you with a choice of which Edition of the software you would like to operate your device. This is the only decision Wave will ask you to make before you can use and enjoy the technology. This option is only presented the first time WaveOS runs on a device or if you decide to go into the settings and switch the edition. Everything concerning configuration and installation before and after this prompt, is fully automatic.

Installation Instructions

Insert Memory

Download the latest version of WaveOS and copy to your Mico SD Card using a tool like Etcher. To use this tool just select the WaveOS Image (dowloadable above) and hit "Flash!". It takes less than 1 minute for WaveOS to install to the SD Card. We suggest a Sandisk (32Gb or higher).

Connect Device

Attach the ethernet Cable to Wave and the other end into your internet router. Then connect the power cord. The system will boot up & use the internet to autonomously set itself up. This process takes 5 - 10 minutes.

Access Menu

Using your smartphone or computer, connect to the Wi-Fi (SSID:Wave-Hotspot/ Password: makeitwave). Once connected enter into your web-browser: "192.168.42.1" or "http://wave/". Then follow the onscreen instructions
